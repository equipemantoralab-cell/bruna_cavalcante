# Drª. Bruna Cavalcante — Landing Page (Google Ads)

Landing page para o fluxo de Google Ads da Drª. Bruna Cavalcante (Reprodução Humana — São Paulo).

## Estrutura

- `index.html` — página única, autossuficiente (CSS e imagens inline em base64)

## Como abrir

Basta abrir o `index.html` em qualquer navegador, ou hospedar como página estática.

## Deploy

- **GitHub Pages**: ative em `Settings → Pages` apontando para `main` na raiz
- **Lovable**: importar este repositório como página/rota no projeto principal
- Qualquer hospedagem estática (Netlify, Vercel, Cloudflare Pages) funciona sem build
